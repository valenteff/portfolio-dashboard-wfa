Portfolio Backtest Dashboard

Interactive analysis of trading strategy performance with outlier detection

Dashboard Overview

This repository contains an interactive dashboard for exploring and analyzing portfolio backtest results.

The dashboard allows you to:

  • Compare different portfolio strategies
  • Analyze equity curves with zooming capability
  • View monthly returns and performance patterns
  • Examine outlier detection and adjustment results
  • Review detailed trading statistics
Deployment Options
Option 1: Deploy on Render (Recommended)

The simplest way to deploy this dashboard is using Render:

Deploy to Render
Option 2: Run Locally

To run the dashboard locally:

git clone https://github.com/valenteff/portfolio-dashboard-wfa.git
cd portfolio-dashboard-wfa
pip install -r requirements.txt
python run_dashboard.py
Option 3: Deploy to Heroku

You can also deploy to Heroku:

heroku create portfolio-dashboard
git push heroku main
